# 🧹 Data Cleaning App 

An interactive web application that allows users to upload datasets and automatically clean them by handling missing values and basic formatting issues.

This project was built as part of my AI & ML learning journey to practice data preprocessing and building interactive Python applications using Streamlit.

---

## 🚀 Demo 

Live App: https://datacleaning-ashley.streamlit.app

---

## 📌 Features 

- Upload CSV or Excel datasets  
- Detect missing values automatically  
- Clean dataset (handle missing values and formatting issues)  
- Download cleaned dataset after processing  
- Basic data preprocessing steps  
- Simple and interactive Streamlit interface  

---

## 🛠️ Tech Stack 

- Streamlit  
- Pandas  
- Openpyxl
